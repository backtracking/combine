ReML

an OCaml library for combinatorics

ReML is an OCaml library for combinatorics problem solving formed by several modules :

  • Dlx: implements Knuth's dancing links
  • Zdd: implements Zero-suppressed binary decision diagrams
  • Emc: a common interface to modules Dlx and Zdd to solve the Exact Matrix Cover problem
  • Tiling: converts a 2D tiling problem into an EMC problem

    • ReML contains as well an interpreter for a language (reml) allowing you to describe tiling problems (like Pentaminoes) and solve them or counting their solutions. With this library you could easily solve other problems like Sudoku or N-queens. The archive contains some exemples of what ReML can do.
